Deploy on Laravel Cloud / Forge

1. Create New Site.
2. Choose HTML.
3. Upload all files from this folder.
4. Make sure index.html is in the site root or public directory.
5. Replace example.com in robots.txt and sitemap.xml with your real domain.
6. Open the site URL and test on mobile + desktop.

Files:
- index.html
- privacy-policy.html
- terms.html
- contact.html
- robots.txt
- sitemap.xml
